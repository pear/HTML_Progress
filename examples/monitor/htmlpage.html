Introduction

This example requires :

  • PEAR::HTML_Progress_Monitor 1.2.0RC3 or better (see source at line 12).

Be aware that:

  • HTML_Progress_Monitor has dependencies (HTML_Page 2.0.0RC2+, HTML_CSS 0.3.4+, HTML_QuickForm 3.1.1+)

Used PEAR::HTML_Page package as QF renderer to display ProgressBar Monitor.
