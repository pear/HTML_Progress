Introduction

This example requires :

  • PEAR::HTML_Progress_Monitor 1.2.0RC3 or better (see source at line 11).

Be aware that:

  • HTML_Progress_Monitor has dependencies (HTML_CSS 0.3.4+, HTML_QuickForm 3.1.1+)

This example will run a default ProgressBar Monitor, and used a default QuickForm renderer without any form template customizations. No user callback was defined.
