Introduction

This example requires :

  • PEAR::HTML_Progress 1.2.0RC3 or better.

This example will run a vertical ProgressBar, beginning at value 85, and increased through 100 with +1 step.

Percent text info is right aligned on 50 pixels width area at right side of the progress bar (default).
Cells have default size and colors.
